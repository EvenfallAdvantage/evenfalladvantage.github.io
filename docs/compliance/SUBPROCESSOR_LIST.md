# Subprocessor List

| **Stripe, Inc.** | Payment processing | Payment card data (tokenized), transaction records, customer billing info | US | Yes (Apr 2026) |
| **GitHub, Inc.** | Source code hosting, CI/CD, static site hosting | Source code, deployment artifacts, CI/CD secrets | US (Azure) | Yes (Apr 2026) |

| **Cloudflare, Inc.** | CDN, DNS, web analytics | IP addresses (anonymized), page view analytics | Global | N/A (minimal data) |
| **OpenStreetMap / Nominatim** | Address geocoding | Address queries (public data, not stored) | EU (community) | N/A (public API) |
